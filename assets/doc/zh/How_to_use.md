# 模拟器用法

设置界面中，开启“使用模拟器”选项，推荐使用MUMU模拟器。

### 模拟器端口号的选择与获取：

#### mumu模拟器：模拟器右上角菜单栏——问题诊断——查找“ADB调试端口”

- 如果使用默认的端口号0，则使用默认建立的初始模拟器

#### 其他模拟器：根据自身情况查找端口号，以下为一些默认端口

- 雷电模拟器 5555
- 蓝迭模拟器 5555
- 逍遥模拟器 21503
- 夜神模拟器 62001 / 59865

### 注意：

- 不同的模拟器，端口号可能不同，需要根据实际情况选择
- 不同的模拟器，可能需要在设置中开启adb端口调试功能，如蓝迭模拟器, 雷电模拟器
- 使用mumu模拟器需关闭“应用保活”功能，否则会导致脚本无法正常运行（已配置自动关闭功能，但可能失效）
- 使用模拟器的时候，需使模拟器的分辨率与AALC中“窗口设置”里的分辨率保持一致
- 模拟器可能会导致脚本运行缓慢，建议使用较高配置的电脑，并给予模拟器足够的内存和CPU资源
